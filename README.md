# pocket-chef-assistant 
# Features
* **Mood-Based Categorization:** Choose between "Manis" (Sweet) or "Asin" (Savory) dishes.
* **Appliance Filter:** Filters recipes based on the tools available in a standard student room (Rice Cooker, Teflon, or Pot).
* **Interactive Cooking Guide:** Step-by-step instructions that interact with the user to check cooking progress or handle cooking failures.

# Tech Stack
* **Language:** Python 3.x
* **Modules:** Built-in `time` module for interactive delays.

# How It Works & Code Structure
The application leverages core programming fundamentals:
* **Functions (`def`):** Modular design where each recipe and the main menu has its own independent function.
* **Control Flow (`if-elif-else`):** Directs the user dynamically based on their inputs.
* **Loops (`while True`):** Implements input validation loops to handle invalid choices and ensure a smooth user experience.
* **Data Structures:** Uses Python lists and `enumerate()` for dynamic numbering of cooking steps.

# How to Run
1. Clone this repository or download the script.
2. Open your terminal or command prompt.
